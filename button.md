## 近いデザインのグループ

- **1, 2, 3** → 同じデザイン（primary pill+icon）のサイズ違い
- **4** → 1〜3に近いが rounded-full ではなく rounded
- **5, 6** → 白pill、ボーダーの有無だけ違う
- **5, 13** → 白pill+border、13はドロップダウン矢印が付く
- **7** → 5に近いが rounded-full ではなく rounded、サイズが小さめ
- **9** → 選択時は5（白pill）に近い見た目だが、emerald-400で切り替わる
- **10, 11, 12** → すべて丸型アイコンボタン（transparent背景+border）、色・サイズ違い
- **14** → StyleDropdownメニューは選択時に primary 背景になる
- **16, 17, 19** → いずれもbg/borderなしのテキスト系
- **5, 20** → 白pill+border、20はborder色がキャラクターカラー
- **1, 2, 3, 21** → primary pill、21はアイコンなし
